## Core Schema

lemon

This data schema provides a standard set of values for Natural Resource Sector core permitting data.

## Scope

The schema is intended to be used in general transactional situations. It comprises of the minimum set of data required to coherently represent a permit and their immediately related attributes.

## Structure

| Name | Cardinality | Type | Description & Constraints |
| --- | --- | --- | --- |
| project_linking_id | 1..1 | `uuid` | A unique key to track all permits related to a project or activity from all permitting systems. |
| project_name | 0..1 | `string` | Short name of the project, e.g., The Hudson, Capitol Hill. |
| TBD | 0..* | `object` | More will be filled in later... |
